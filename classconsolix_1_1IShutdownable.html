Detailed Description

Interface for components supporting shutdown logic.

This interface defines a method for handling shutdown events, which is particularly useful for releasing resources, saving state, or performing cleanup operations when a termination signal is received. Access to the shutdown method is restricted to the AppComponentManager class.

Handles shutdown logic for the component.

This method is called when a shutdown signal is received. Derived components should implement this method to perform necessary cleanup or save state before termination.
